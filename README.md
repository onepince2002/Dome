## 📋 專案簡介

這是一個簡單的測試專案，包含了基本的 HTML 網頁和 JavaScript 檔案。專案主要用於學習和測試 GitHub 的基本功能。

## 📁 檔案結構

```
Dome/
├── README.md       # 專案說明文件
├── index.html      # 主要 HTML 網頁檔案
└── test.js         # JavaScript 測試檔案
```

## 🚀 功能特點

- **簡單的 HTML 頁面**：包含基本的中文網頁結構
- **JavaScript 測試**：包含 console.log 輸出和 alert 彈窗
- **UTF-8 編碼支援**：完整支援中文字元顯示

## 📦 安裝與使用

### 方法一：直接開啟

1. 克隆此專案到本地：
```bash
git clone https://github.com/onepince2002/Dome.git
```

2. 開啟 `index.html` 檔案：
   - 直接雙擊 `index.html` 檔案
   - 或使用瀏覽器開啟此檔案

### 方法二：使用網頁伺服器

使用 Python 啟動簡單的本地伺服器：

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然後在瀏覽器中訪問 `http://localhost:8000`

### 測試 JavaScript

開啟瀏覽器的開發者工具（F12），在控制台中可以看到：
```
Hello from test.js!
```
